Reproduction Instructions

Note: This page is for development and tracking purposes only. For end-user setup instructions, please refer to the Setup tab on the home page.

Development Process

The following steps were undertaken to set up and reproduce the project environment:

  1. Install Ruby Development Tools: 
    sudo apt-get install ruby-dev
  2. Install Jekyll: 
    gem install jekyll bundler
  3. Create Jekyll Site: 
    jekyll new docs
  4. Create Custom HTML Files:
    • Default Layout: Created a new layout file in _layouts/default.html.
    • Additional Pages: Created individual HTML files (e.g., overview.html, setup.html, reproduction.html).

Reproducing This Environment

To reproduce this environment on your local machine:

  1. Clone the Repository: 
    git clone https://github.com/yourusername/DrawGPT.git
  2. Navigate to the docs Directory: 
    cd docs
  3. Install Dependencies: 
    bundle install
  4. Run Jekyll Server: 
    bundle exec jekyll serve
  5. Access Locally: Open http://localhost:4000 in your browser.
